1000-Introduction


INTRODUCTION TO SymShell

SYMSHELL is a DOS based commandline shell. Which can be run in a window or full screen.

1100-Feature overview


FEATURES

You can execute shell applications. To see a detailed help type HELP or HELP command to display the help of a symshell command.


2000-Using SymShell


Using SymShell

Type [FULL] + [ENTER] to switch between fullscreen or window mode (or GRAPH+ENTER on MSX) [TAB] to autofill [HELP] to display help [HELP COMMAND] to display help about a command or [COMMAND %?] to display help about a command.

Using different sizes and colors in SymShell
You can change the background color and text color in the setting menu of symshell. If you want to change the size of SymShell type SIZE X Y like SIZE 80 24 gives 80 characters on one line and 24 lines. Be aware when using shell applications like telnet you choose the correct size before you start telnet.
